A `.deb` package has been provided that will install all of the templates onto an Ubuntu or Debian-based operating system for immediate use in LibreOffice Impress.

* Get the Ubuntu install package [**here**](https://github.com/dohliam/libreoffice-impress-templates/releases/download/v2.2/libreoffice-impress-templates-all_2.2-1.deb)

This guide will take you step-by-step through the process of installing the package on your system.

1. Download the .deb package from the link above.
2. Install the package. There are two ways to do this, graphically and through the terminal, which will be outlined separately below:
3. ***Command-line install***: 1. Open the terminal (e.g. the `terminal` or `konsole` etc) and change to the directory where you downloaded the file. For example, if you downloaded the file to your `Downloads` folder, enter `cd ~/Downloads` 2. Enter the following command to install the package: `sudo dpkg -i libreoffice-impress-templates-all.deb`
4. ***Graphical install***: 1. Open your file manager (e.g. Nautilus, Dolphin, Thunar, PCManFM, Konqueror, etc) and navigate to the downloaded package file (if you didn't specify a download location, it's probably in your `Downloads` folder) 2. Double-click on the file to open the default package installer (if this doesn't work, you could also try right-clicking on the file and then selecting "Open with..." and choosing a package installer such as `gdebi` or `QApt Package Installer`) 3. Click on the "Install Package" button and follow the prompts to install the package (this may ask you to enter your password)
5. Open LibreOffice Impress
6. Open the Template Manager: *File > Templates > Manage* or `Ctrl+Shift+N`
7. Click on the *settings > Refresh* button at the bottom left of Template Manager window.
8. Click on the *Presentations* tab at the top of the Template Manager window. You should see 200+ new templates listed.
